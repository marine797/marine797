# 여행말문 300 — 영어/일본어 여행 회화 앱 전체 코드

React + TypeScript + Vite 기반 모바일 웹앱입니다.

## 포함 기능

- 영어 / 일본어 전환
- 영어 데이터 259개, 일본어 데이터 259개 포함
- 카테고리 선택
- 검색
- 천천히 / 보통 / 빠르게 듣기
- 짧은 표현 듣기
- 긴 표현 듣기
- 카테고리 반복 듣기
- 정지 버튼
- 일본어는 ja-JP, 영어는 en-US, 한국어 뜻은 ko-KR 음성으로 재생
- 50+ 시니어가 쓰기 편한 큰 글씨 / 큰 버튼 UI

## 실행

```bash
npm install
npm run dev
```

## 배포용 빌드

```bash
npm run build
```
